Document Status and Features

This iteration is housed in the Silver Fox folder. Prototype 08 was built after prototype 09, after deciding to continue using BS4.

This iteration utilizes BS4 styles added using cwebba_bootstrap_0503b2.css.

The CSS stylesheet name is style_0703_v8.css 070316.

Select Drop-down navigation styles from Pegasus, which used BS03, are in bs-3_nav.css.

Final drop-down styles are provided by sisiharmoni_bs04_070416.css

These styles will need to be assimilated into sisiharmoni-bs05 SASS, with SilverFox styles. Styles to control the Logo in bs-3_nav.css have been removed.

Responsive-tabs has been added to aid with testing sticky footer on iPhone.

The prototype is based on http://codepen.io/paulobrien/pen/FKAxH, titled "Ultimate Sticky footer IE8+ (no hacks)".

During testing it was found that the .topmast declaration, "display:table;", in style_sans_BS-0503b.css conflicted with the sticky-footer. Changing it to "display:table-cell;" fixed the problem. The declaration is removed in style sheet style_0703_v8.css. The header has a "display:table-cell;" declaration in the HTML styles.

BS03 Drop-down navigation styles (070116) was modified. The .navbar-header {} declaration is modified. New .in-head and .flex-vessel declarations were added to style_sans_BS-0503b2.css, 070116, and inner elemeents associated media query and declaration are modified. The backgroud image is contained in a wrapper ".in-head" div. A third wrapper is set to flexbox @768px.
